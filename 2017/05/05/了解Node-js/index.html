了解Node.js

5月 05, 2017 发布在 服务器

Node.js作为一个JavaScript编写的web框架，其中有很多方便使用的地方，在此记录一下

Modoules

node.js 常用的框架list

  • express

cnpm install express --save

  • body-parser

node.js中间件, 用于处理JSON, Raw, Text和URL编程的数据。
cnpm install body-parser --save

  • cookie-parser

一个解析Cookie的工具。 通过req.cookies可以取到传过来的cookie, 并把他转成对象

cnpm install cookie-parser --save

  • multer

node.js中间件, 用户处理enctype="multipart/form-data"(设置表单的MIME编码)的表单数据.

cnpm install multer --save

理解Swift的Copy-on-write机制

 2018年1月7日

翻译：原文链接：Understanding Swift Copy-on-Write mechanisms

在Swift中，我们有引用类型(类Classes)和值类型(结构体Structs, 元祖Tuples, 枚举enums)。值类型有一个Copy机制: 如果你将一个值类型变量赋值给你一个变量，或者将它作为参数传递给一个函数(inoout 参数除外), 这个值类型的数据将会被Copy。你将会有两个相同内容的值，同时被声明了两个不同的内存空间。关于引用类型和值类型的区别，请参赛Apple官方内容。
